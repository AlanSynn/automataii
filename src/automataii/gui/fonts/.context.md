# Context: src/automataii/gui/fonts/

This directory, `fonts/`, within the `gui` sub-package, is designated for storing custom font files (e.g., `.ttf` - TrueType Font, `.otf` - OpenType Font) that are used by the `automataii` application's Graphical User Interface.

## Purpose:

- **Consistent Typography**: To ensure a consistent visual appearance of text elements across different operating systems and user setups, regardless of which fonts are installed locally on a user's machine.
- **Specific Branding/Design**: To use specific typefaces that are part of the application's visual design, branding, or style requirements.
- **Special Characters/Symbols**: To include fonts that provide special characters, icons, or glyphs not commonly available in standard system fonts.

## Current Contents:

- **`Segoe UI.ttf`**: Standard Segoe UI font.
- **`Segoe UI Italic.ttf`**: Italic variant of Segoe UI.
- **`Segoe UI Bold.ttf`**: Bold variant of Segoe UI.
- **`Segoe UI Bold Italic.ttf`**: Bold Italic variant of Segoe UI.

(This indicates the application intends to use the Segoe UI font family for its interface.)

## Integration with PyQt6/Qt:

Fonts in this directory need to be programmatically loaded into the application using Qt's font database, typically at application startup. This is done using `QFontDatabase.addApplicationFont()` or `QFontDatabase.addApplicationFontFromData()`.

Example (conceptual, usually in `main.py` or `app/init.py` or `gui/main_window.py` initialization):
```python
from PyQt6.QtGui import QFontDatabase
import os

# Construct the path to the fonts directory (adjust relative path as needed)
# This example assumes this code might be run from a script within src/automataii/gui/
# or that the path is correctly resolved from the main application entry point.
fonts_dir = os.path.join(os.path.dirname(__file__), "fonts")

if os.path.isdir(fonts_dir):
    for font_file in os.listdir(fonts_dir):
        if font_file.lower().endswith( (".ttf", ".otf") ):
            font_path = os.path.join(fonts_dir, font_file)
            font_id = QFontDatabase.addApplicationFont(font_path)
            if font_id == -1:
                print(f"Warning: Could not load font: {font_path}")
            else:
                font_families = QFontDatabase.applicationFontFamilies(font_id)
                # print(f"Successfully loaded font: {font_path} (Families: {font_families})")
```

Once loaded, these fonts can be referenced by their family name when setting widget styles (e.g., via Qt Style Sheets - QSS) or when creating `QFont` objects directly in code.

## Developer Guidelines:

- **Licensing**: CRITICAL - Ensure that all font files included in this directory have licenses that permit their distribution with the application. Include license files or references if required by the font vendor.
- **File Formats**: Prefer widely supported font formats like TTF or OTF.
- **Loading**: Implement robust font loading at application startup. Log any errors if fonts fail to load.
- **Fallback**: Consider font fallback strategies in your UI design in case custom fonts cannot be loaded or rendered correctly on a specific system, although `addApplicationFont` aims to make them available.
- **Minimization**: Only include font variants (regular, bold, italic, etc.) that are actually used by the application to keep the package size down.
