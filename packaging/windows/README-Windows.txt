MotionSmith for Windows
=======================

1. Extract MotionSmith-windows.zip.
2. Double-click MotionSmith.exe.

Do not run MotionSmith.exe from inside the zipped folder; extract the zip first.

Note: research/test builds signed with a self-signed certificate may still show a Windows SmartScreen warning. Use a CA-issued code-signing certificate for trusted public distribution.
