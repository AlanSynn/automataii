# Automataii Project Overview

## Purpose
Automataii is an advanced mechanism design, simulation, and animation platform with real-time parametric editing capabilities. It combines kinematics simulation with intuitive direct manipulation interfaces for mechanical system design.

## Key Features
- **Mechanism Design**: Interactive design of 4-bar linkages, cam systems, and gear trains
- **Parametric Playground**: Real-time manipulation of mechanism parameters through drag-and-drop handles
- **Kinematic Simulation**: Advanced forward/inverse kinematics with collision detection
- **Animation Pipeline**: Character animation from static drawings with mechanism-driven motion
- **Modular Architecture**: Event-driven, dependency-injected architecture for extensibility
- **Project Management**: Compressed .atii project format with version control

## Application Type
Desktop application built with PyQt6 (cross-platform Qt GUI framework)

## Target Users
- Mechanical engineers
- Researchers
- Animation designers
- Anyone interested in understanding how mechanical systems move
