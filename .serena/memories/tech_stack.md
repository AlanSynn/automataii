# Automataii Tech Stack

## Language
- Python 3.13+ (requires-python = ">=3.13")

## Package Management
- **uv** - Modern Python package manager (NOT pip!)
- Uses pyproject.toml for dependency management

## Core Dependencies
- **PyQt6** (6.8.0+) - GUI framework
- **NumPy** (2.0.0+) - Numerical computing
- **OpenCV** (4.11.0+) - Computer vision
- **ONNX Runtime** (1.22.0+) - ML model inference
- **SciPy** (1.15.0+) - Scientific computing
- **scikit-image** (0.25.0+) - Image processing
- **scikit-learn** (1.7.0+) - Machine learning
- **Matplotlib** (3.9.0+) - Visualization
- **Pydantic** (2.11.0+) - Data validation
- **PyYAML** (6.0.2+) - Configuration files
- **Pillow** (10.0.0+) - Image manipulation
- **python-constraint** (1.4.0+) - Constraint solving

## Development Tools
- **pytest** (8.0.0+) - Testing framework
- **ruff** (0.8.0+) - Linting and formatting
- **mypy** (1.13.0+) - Type checking
- **black** (24.0.0+) - Code formatting
- **pre-commit** (4.0.0+) - Git hooks

## Build System
- **hatchling** - Build backend
- **PyInstaller** - Creating executables for different platforms
