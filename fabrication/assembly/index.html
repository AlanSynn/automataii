Automataii board assembly

15 x 15 means 225 board holes. Use the row letter and column number on the board map.

Quick start

  1. Print or fabricate the linked part templates.
  2. Open board-15x15.svg and find the called-out holes.
  3. Put the paper fastener through the board, add spacers, add the part, then leave the tabs loose.
  4. After each step, run the motion check before adding the next layer.
